# FarmTech Solutions

Sistema de Agricultura Digital desenvolvido em Python e R.

## Funcionalidades

- Cálculo de área de plantio
- Cálculo de manejo de insumos
- Análise estatística com R
- Integração com API meteorológica

## Tecnologias utilizadas

- Python
- R
- GitHub

## Estrutura do projeto

python/ -> sistema principal  
R/ -> análises estatísticas e API  
docs/ -> resumo do artigo  
video/ -> link da apresentação
